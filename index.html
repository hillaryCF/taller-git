Que es JQuery

JQuery es una librería de JavaScript . Esta librería de código abierto, simplifica la tarea de programar en JavaScript y permite agregar interactividad a un sitio web sin tener conocimientos del lenguaje. jQuery es la biblioteca de JavaScript más utilizada.
